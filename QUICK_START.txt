📋 STEP-BY-STEP SETUP
─────────────────────────────────────────────────────────────────────────────

1. INSTALL DEPENDENCIES (First time only)
   ────────────────────────────────────────
   cd "C:\Users\Admin\Desktop\Coding Stuff\image-detection-app"
   pip install -r requirements.txt

   This installs:
   • mediapipe (hand detection) ← NEW!
   • opencv-python (camera & display)
   • tensorflow, numpy, scipy, requests

   Takes 2-3 minutes...


2. RUN THE ASL APPLICATION
   ────────────────────────
   python src/asl_main.py

   Expected output:
   • Console messages showing initialization
   • Camera window opens showing live feed
   • Green hand landmarks appear when hands visible
   • Letter detection with confidence scores


3. TEST WITH HAND SIGNS
   ────────────────────
   Start with easiest letters:

   ⭐ LETTER A (Easiest)
      • Make closed fist
      • Extend thumb to side
      • You'll see: "A: 0.85 (Right)"

   ⭐ LETTER L (Very Clear)
      • Extend index finger up
      • Extend thumb to side
      • You'll see: "L: 0.86 (Right)"

   ⭐ LETTER V (Very Clear)
      • Extend index & middle fingers
      • Spread them apart
      • You'll see: "V: 0.87 (Right)"

🔍 TROUBLESHOOTING QUICK TIPS
─────────────────────────────────────────────────────────────────────────────

Problem: Hand not detected
  → Check lighting (use natural light or lamp)
  → Get closer to camera (6-12 inches away)
  → Ensure entire hand is visible

Problem: Wrong letter recognized
  → Make hand pose more distinct/clear
  → Hold pose steady for 1-2 seconds
  → Try different hand angle
  → Check lighting (no shadows)

Problem: Low confidence (<0.75)
  → Improve lighting
  → Hold hand more steadily
  → Make more exaggerated poses

Problem: Landmarks not showing
  → Press 'l' key to toggle on
  → Restart application
  → Check MediaPipe installed: pip install mediapipe
